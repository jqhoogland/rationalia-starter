# Tool AI

**Tool AI** is a type of [[Artificial intelligence|artificial intelligence]] that resembles a tool rather than an agent. Tool AIs are claimed to pose fewer risks than agent AIs, and are for this reason proposed as a possible solution (or dissolution) of the [[AI alignment|AI alignment problem]].
